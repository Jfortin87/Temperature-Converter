# A Simple Temperature Converter

- **Branch**: Main

This is a simple temperature converter that converts Celsius to Fahrenheit and vice versa. It is built using HTML, CSS, and JavaScript. The user can input a temperature in either Celsius or Fahrenheit and click the corresponding button to see the converted temperature.

### Purpose:

- Beginner practice project to learn how to use JavaScript to manipulate the DOM and perform calculations.
- Simple and fun project to learn how to convert temperatures between Celsius and Fahrenheit.
- This project is also a good way to practice using HTML and CSS.
- This project is also a good way to practice using GitHub and version control.
-

## Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius

### Technologies Used:

- **HTML:**
  - The structure of the webpage, including the input fields, buttons, and containers for displaying results.
  - The HTML file defines the layout and elements of the temperature converter, such as the input fields for Celsius and Fahrenheit, the buttons to trigger the conversion, and the containers to display the results.
  - The HTML file also includes links to the CSS file for styling and the JavaScript file for functionality.
  - Tags, Classes, ID, Forms, Buttons

- **CSS:**
  - The styling of the webpage, including the layout, colors, fonts, and overall design.
  - The CSS file defines the visual appearance of the temperature converter, such as the background color, font styles, borders, and spacing.
  - The CSS file also includes styles for the input fields, buttons, and result containers to make them visually appealing and user-friendly.
  - Selectors, Properties, Values, Box Model, Flexbox
- **JavaScript:**
  - The functionality of the webpage, including the logic for converting temperatures and updating the DOM with the results.
  - The JavaScript file contains functions that perform the temperature conversion calculations and update the content of the result containers based on user input.
  - The JavaScript file also includes event listeners for the buttons to trigger the conversion when clicked.
  - Variables, Functions, Event Listeners, DOM Manipulation
- **GitHub:**
  - Version control and hosting for the project.
  - The GitHub repository allows for tracking changes to the code, collaborating with others, and sharing the project with the public.
  - The repository includes the HTML, CSS, and JavaScript files, as well as a README file that provides information about the project.
